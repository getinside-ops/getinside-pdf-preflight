Industry detection — which approach?

How should the tool handle industry once we can auto-detect it?

A

Silent auto-select — no user input

Industry is detected and used automatically. The detected value shows in the results summary, but the user never has to think about it before running.

Pros

  • Zero friction — upload and go
  • Fewer fields = cleaner form

Cons

  • If detection is wrong, results are silently wrong
  • No override before analysis
B

Auto-detect + visible override (recommended)

After upload and analysis, the industry is auto-detected and shown as a badge in the results. The user can click to override if wrong — re-runs only the industry check instantly.

Keep dropdown — pre-fill the detected value

The industry dropdown stays exactly as-is, but it's pre-populated with the detected industry. User confirms or changes it before running analysis.
